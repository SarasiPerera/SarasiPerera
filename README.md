## 🚀 Featured Projects

### 🧒 SmartKid Insight *(Final Year Research Project)*
> Adaptive micro-interaction game framework for preschool skill assessment — targeting Sinhala-speaking children aged 3–6.

Embeds **stealth assessment** in gameplay so children are evaluated without explicit testing. Uses **Random Forest** to classify skill levels (Strong / Developing / Weak) and an **ANN-based recommendation engine** to serve personalised next activities. Built with Flutter + Python/Flask + Firebase following DSRM.

`Flutter` `Python` `Flask` `Firebase` `scikit-learn` `Random Forest` `ANN` `DSRM`
